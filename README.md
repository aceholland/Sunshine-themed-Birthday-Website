Sunshine Birthday Scrapbook
A deeply personal, handmade-feeling digital scrapbook built as a birthday gift. Sunshine-core aesthetic, golden hour energy, warm nostalgia, and cozy journal vibes, all in a single self-contained HTML file.

Whats inside?

Landing Page : Animated sunrays, floating bokeh, hero polaroid, cinematic intro text

Sunshine Memories : Draggable polaroids with sticky notes, golden-hour photo treatment

Tiny Warm Notes : Scattered handwritten-style cards with typewriter reveal on scrollReasons

Why You Matter: Tap-to-expand sticky note cards

Music Corner : Spinning vinyl disc player with track list

Timeline Journey: Scrapbook diary pages connected by a glowing dotted path

Final Letter: Envelope with 3D flap-open animation, letter slides out

Hidden Easter Eggs : 6 clickable secrets scattered across every section

Finale Overlay: Full-screen golden burst with floating symbols when all is read

 How to Use

Open birthday-scrapbook.html in any modern browser — no server needed
Add photos by placing images in an assets/images/ folder next to the HTML file:

Personalize — search for birthday in the file and write the recipient's name after it
Send it — zip the HTML file + assets/ folder, or host it on GitHub Pages / Netlify for a shareable link


If images are missing, warm emoji placeholders appear automatically — it still looks great.


✏️ Customization
Change the name:
html<!-- In the HTML, find and replace: -->
Happy Birthday,   →  Happy Birthday, [Name]
Dear Friend,           →  Dear [Name],
Edit the letter — find the #final-letter section and rewrite the <p> tags inside .letter-paper
Edit memories & notes — all text is plain HTML, easy to find and change in any text editor
Add real music — replace the track list items with actual <audio> elements pointing to your .mp3 files

🛠️ Technical Details

Single file — all CSS and JS are inlined, zero dependencies, zero build step
No frameworks — vanilla HTML, CSS, and JavaScript only
Fonts — loaded from Google Fonts (requires internet on first load; cached after)
Animations — CSS keyframes + IntersectionObserver for scroll triggers
Particles — Canvas-based floating golden symbols (✦ ✿ ☀ ♥)
Responsive — works on mobile and desktop
